#  DevBank

## 📜 Descrição
Projeto que realiza as validações necessárias para o cadastro e login de um usuário, e facilita o preenchimento do endereço com dados obtidos da API [ViaCep](https://viacep.com.br/).

## 🖥️ Tecnologias
As seguintes ferramentas foram usadas na construção do projeto:

🛠 Kotlin<br>
🛠 Arquitetura MVVM<br>
🛠 Coroutines<br>
🛠 Retrofit<br>
🛠 Logging Interceptor - OkHttp<br>
🛠 Injeção de Dependência com Koin<br>
🛠 Shared Preference<br>
🛠 Jetpack Navigation
